# 🏃 体育理论题库查询系统

一个美观、快速的体育理论考试题库查询工具。

## ✨ 功能特点

- 🔍 **快速搜索**：支持关键词模糊搜索，毫秒级响应
- 📚 **海量题库**：包含 60,000+ 道题目
- 🎨 **优美界面**：现代化设计，支持明暗主题切换
- 📝 **题型筛选**：可按判断题/选择题筛选
- 📄 **分页浏览**：结果分页显示，浏览更便捷

## 🚀 使用方法

### 方式一：双击启动
直接双击 `启动题库查询.bat` 即可运行程序。

### 方式二：命令行启动
```bash
.venv\Scripts\python.exe main.py
```

## 📖 使用说明

1. **搜索题目**：在搜索框输入关键词，按回车或点击搜索按钮
2. **筛选类型**：使用下拉菜单选择"判断题"、"选择题"或"全部"
3. **搜索选项**：勾选"同时搜索选项内容"可在选择题的选项中搜索
4. **翻页浏览**：使用底部的分页按钮浏览更多结果
5. **切换主题**：点击右上角的 🌙/☀️ 按钮切换明暗主题

## 📁 文件说明

- `main.py` - 主程序（图形界面）
- `question_parser.py` - 题库解析模块
- `questions.json` - 解析后的题库数据
- `启动题库查询.bat` - Windows 快捷启动脚本
- `体育理论考试总题库.docx` - 原始题库文件

## 🔧 重新解析题库

如果需要更新题库，可以运行解析模块：

```bash
.venv\Scripts\python.exe question_parser.py
```

## 💻 技术栈

- Python 3.11
